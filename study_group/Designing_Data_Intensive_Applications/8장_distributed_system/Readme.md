## 2G -> TDMA , FDMA

TDMA (Time Division Multiple Access)

- 주파수 하나를 사용자에게 고정하긴 하는데, 시간별로 받아갈 사용자가 다르다. 
- a 사용자는 offset 0초에서 0.577ms마다 보내는거고, b 사용자는 offset 0.577ms에서 0.577ms마다 보낸다. 
- 예를들면, 0.1초마다 나누고, 한 회선에 8명까지 들어갈 수 있다고 치면, 0.8초 이후의 패킷을받는 셈

FDMA (Frequency Division Multiple Access)

- 주파수를 A는 900MHz, B는 901MHz 사용하게 해서 패킷 교환.

TCP는 전화와 다르게, opportunistic - 가변크기의 데이터블럭을 최대한 빠르게 보내고, 안쓸땐 아예 안써. 

패킷 교환을 쓰는 이유는, bursty traffic에 최적화되어서.

QoS: Quality of Service : 패킷에 우선순위를 매기고 스케줄링
