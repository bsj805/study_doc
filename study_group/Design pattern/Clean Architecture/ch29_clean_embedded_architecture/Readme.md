# 29. 클린 임베디드 아키텍쳐

- 소프트웨어는 닳지 않지만, 펌웨어와 하드웨어에 대한 의존성을 관리하지 않으면 파괴된다. (수명이 짧아진다.)
- 펌웨어는 개별 하드웨어에 맞추어 내장되는 소프트웨어이다.
- 나도 코드에 SQL을 심어 놓거나 개발하는 코드 전반에 플랫폼 의존성을 퍼뜨려 둔다면 펌웨어를 작성하는 셈이다.
