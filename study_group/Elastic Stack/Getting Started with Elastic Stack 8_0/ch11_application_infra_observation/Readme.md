# 주요 관측지표

- 메트릭
  - 시간 경과에 따른 상태 및 성능 지표 
- 로그
  - 특정 시점에 발생하는 이벤트 기록 
- 추적(trace)
  - 함수 호출과 스택을 추적
  - 외부 서비스로 보내는 요청 
  - 코드 실행중 오류와 예외 
  

# 합성 모니터링, 실제 사용자 모니터링 

Real User Monitoring : RUM 에이전트를 사용하면 페이지 로드 지연 시간, DOM 렌더링 성능 등까지 데이터 제공 가능. 

Uptime monitoring : 가동시간, 가용성 , availability . 잘 처리되는지랑은 다른 

합성 모니터링 : 사용자처럼, 모니터링하고 싶은 주요 작업흐름을 따라가는 모니터링
